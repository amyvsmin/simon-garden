# Substack 新手入門指南｜Janis Zeng（從 0 到 2070 訂閱、6 個月複利）

## 摘要

Janis Zeng 把自己 Substack 從 0 訂閱到 2070 訂閱、過程花 10 個月（前 3 個月寫沒人看、找對方法後 6 個月內到 2070）的方法整理成 5 部分指南：為什麼選 Substack、兩個核心功能（長文 + Notes）、找寫作主題的 4 切入點、可持續寫作節奏、站內自然曝光 3 戰術。核心主張是「沒廣告預算、不跨平台宣傳、每天 15–30 分鐘」靠站內機制就能成長；前提是把 Substack 當「自有受眾 + 內容資產」雙複利平台、配雙標準節奏避開倦怠。

## 核心概念

- [[substack]]：訂閱制電子報結合永久頁面的雙軌觸達平台、4 個結構性優勢（自有名單／內容長壽／先發優勢／介面簡潔）
- [[owned-audience]]：擁有可匯出 email 名單作為平台中立、可帶著走的長期資產；訂閱制不靠演算法
- [[content-asset]]：每篇文章是會持續為你工作的資產、3 個月前的文今天仍能帶新訂閱；對比 X／FB 動態的 1–6 小時壽命
- [[substack-notes]]：站內最主要曝光機制；3 類型（解決方案／觀點／旅程）；長文發布後寫 Note 引導點擊
- [[substack-resurfacing]]：每篇文章至少 3 次曝光（發布日 Note 推 / 一週後 Restack / 任何時候精選引言）；同篇可 Restack 兩次
- [[sustainable-writing-cadence]]：複利要 6 個月、雙標準（最低每週 1 篇 + 2 Notes / 理想每週 2 篇 + 5 Notes）、固定流程取代等靈感、允許休息

## 對 Simon 的應用

**短期（本週內）**：
1. CYBERSEC Day 1 已發 5/7 早上、按 [[substack-resurfacing]] 戰術今天寫一則 Note 推廣（解決方案型 / 觀點型擇一）；5/14 左右做第一次 Restack；之後挑「8 秒武器化」段落做精選引言
2. 修正 [[substack]] 上 Day 1 文章的 rendering 問題（## headings + --- 分隔線、見 `project_pending_cybersec_day1_substack_fixes.md`）

**中期（接下來 1 個月）**：
1. CYBERSEC Day 2 / Day 3 系列接著發、套用 [[sustainable-writing-cadence]] 雙標準（先設「每週 1 篇」最低標）
2. 每篇套 [[substack-resurfacing]] 3 次曝光標準流程
3. 用 [[substack-notes]] 3 類型輪流發、找出 Simon 自己風格最自然的類型

**長期（3–6 個月）**：
1. 撐過第一個月「沒人看」的低谷期、按 6 個月複利轉折點預期
2. 累積 ≥ 6 篇後評估升級到 cookie API 自動發布（見 `~/.claude/user-memory/reference_substack_markdown_paste.md` 方案 C）
3. 訂閱數穩定後再考慮 Medium / LinkedIn 跨平台、依然以 Substack 為主漏斗

**注意點**：不分散到 IG／Threads／FB 多平台、作者 90% 訂閱來自 Substack 站內；先在這把根扎深。

## 原文要點

### 第一部分：為什麼選 Substack
- 你在社群平台上的追蹤者不屬於你（演算法、平台規則改變即觸及歸零）
- Substack 4 優勢：擁有訂閱者清單可匯出 / 內容壽命長被 Google 搜尋到 / 月活 2000–3000 萬仍屬早期中段市場 / 介面簡潔不需架站
- 作者實測：前 3 個月迷茫 → 找對方法後 6 個月到 2070 訂閱 / 每天 10–30 分

### 第二部分：兩個核心功能
- **長文**：建立深度信任、發布即寄 email + 永久頁面被搜尋；每週 1–2 篇固定發布；起步全免費再考慮付費牆；500 / 1000 / 2000 字都算長文，重點深度
- **Notes**：站內主要曝光機制；3 類型解決方案／觀點／旅程；節奏初期 2–3 天 1 則、熟悉後每天 1 則；長文發完寫一則 Note 引導點擊

### 第三部分：找寫作主題 4 切入點
- 從專業出發（讀過 3 本以上書 = 比 90% 一般人懂）
- 從興趣出發（自己喜歡寫得久、不容易倦怠）
- 記錄學習過程（把昨天的自己當讀者、邊學邊寫）
- 寫你的故事（人對故事的感受 > 知識）
- 起步方法：花 10 分鐘回答 3 問題：最近在思考／學什麼？常被問什麼？我自己想讀什麼？

### 第四部分：可持續寫作節奏
- 複利需要時間（6 個月才看到效果、多數人撐不到）
- 最低 / 理想雙標準（最低每週 1 篇 + 2 Notes / 理想 2 篇 + 5 Notes）
- 流程化（固定時段 + 儀式 + 低門檻 50 字也算）
- 允許休息（一週不發也 OK、回來繼續出現）

### 第五部分：站內自然曝光 3 戰術
- Note 推廣（發布當天寫 Note 引導）
- 重新堆疊 Restack（舊文重新加回 Notes 動態牆、同篇可推兩次）
- 精選引言 Quote Note（選段落自動生引言 Card）
- 標準流程：每篇 ≥ 3 次曝光（發布當天 / 一週後 / 任何時候）
- 跨平台推廣（IG／Threads／X）效益不大、作者 90% 訂閱來自站內
